CODE-MIXED QUESTION ANSWERING CHALLENGE

Brief description about the web-site:

If we observe conservation between people in social media, The messeges are being posted in mixed-level languages almost 2000-2100 different lnguages

Here, We worked on three different mixed-level languages such as Telugu+English, Tamil+English and Hindi+English

Some examples of mixed-level language questions: Input: Output:

(Telugu+English) Input: Taj Mahal evaru nirminchaaru Output: The answer type is "PERSON", The predict answer is Jahan
(Hindi+English) Input: narendra modi india ke pradhaana mantree hain? Output: The answer type is "YESNO", The predict answer is belongs to YES or NO
(Tamil+English) Input: Eiffel Tower endha natla iruku Output: The answer type is "LOCATION or PLACE", The predict answer is Paris, Poland

Other information about the project:

The Project is totally based on NLP
Natural language processing, NLP is a branch of artificial intelligence that deals with analyzing, understanding and generating the languages that humans use naturally in order to interface with computers in both written and spoken contexts using natural human languages instead of computer languages
